CSV files must be encoded in UTF-8 (BOM optional). They may contain the following columns, in any order: Person Number (optional), Given Name (required), Family Name (required), Country Code (required), Primary Role (required), Other Roles (optional), Guide For Codes (optional), Allergies and Dietary Requirements (optional), Phone Number (optional), Photo (optional), Event Photos Consent (optional), Photo Consent (optional), Allergies and Dietary Requirements Consent (optional). In addition, they may contain any number of columns with names ‘Contact Email 1’, ‘Contact Email 2’, etc., giving contact email addresses for each person. Person numbers are the numbers in the for previous participation, or empty for a person who has not previously participated. If ‘Photo’ is specified, it must be the filename of a photo in a ZIP file uploaded at the same time (in exactly the same form as the filename appears in the ZIP file, including any directory components; for example, ‘dir/photo.jpg’). ‘Photo Consent’, if specified, must be one of ‘not_applicable’, ‘no’, ‘badge_only’, ‘yes’. Other consent columns, if specified, must be one of ‘Yes’ and ‘No’. Unknown columns are ignored. Only participants for staff countries may be registered through bulk registration. If a contact email address is specified, a self-registration account will be created for that person to allow them to complete other registration details.
